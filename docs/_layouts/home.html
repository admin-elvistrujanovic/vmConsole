vmConsole is a free and open-source application that enables you to run Alpine Linux distribution in a virtual machine on your Android device. Thousands of packages available for installation from online repositories make vmConsole a powerful tool for software developers, testers, system administrators and just Linux fans. The core of application is QEMU x86_64 emulator, so you do not need superuser permissions, KVM support or other special features of device. Input and output are being performed through Xterm-compatible terminal screen.
